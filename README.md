### Expense Tracker


## Project Overview
The Expense Tracker is an application that helps users manage their daily expenses. Users can add expenses, categorize them, view all recorded expenses, remove unwanted entries and calculate the total amount spent.

## Features

* Add new expenses with description, amount and category.
* Display all expenses in a structured list.
* Delete expenses.
* Calculate and display the total expenses.
* Filter expenses by categories.
* Save expenses using Lostorage.
* Load saved expenses when the application starts.

## Technologies used
- HTML
- CSS
- JavaScript
- Jest

## Usage
1. Enter the expens description
2. Enter the expense amount
3. Select a category
4. Click the add expense button
5. View all recorded expenses
6. Delete expenses when necessary
7. View the total amount spent
